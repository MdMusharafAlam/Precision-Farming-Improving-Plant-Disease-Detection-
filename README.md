Plant diseases significantly reduce agricultural productivity. This project uses Machine Learning / Deep Learning techniques to detect plant diseases from leaf images and provide accurate predictions.

The system allows users to upload an image of a plant leaf, and the model predicts whether the plant is healthy or affected by a specific disease.

Features

🌿 Upload plant leaf images
🤖 Automatic disease prediction using trained ML/DL model
📊 High accuracy prediction
🧠 Trained using image dataset
🌐 Simple and user-friendly interface
⚡ Fast and efficient processing

Technologies Used

Python
TensorFlow / Keras (for deep learning model)
OpenCV (for image processing)
NumPy & Pandas
Flask / Streamlit (for web application)
